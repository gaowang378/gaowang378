    基础选择器：

    基础选择器 作用 特点 使用情况 用法
    标签选择器 可以选出所有相同的标签 不能差异化选择 较多 p{color:red}
    类选择器 可以选出1个或者多个标签 可以根据需求选择 非常多 .nav {color:red}
    id选择器 一次只能选择一个标签 ID属性只能在每个HTML文档中出现一次 一般和js搭配 #nav{color:red}
    通配符选择器 选择所有的标签 *{color:red}

    字体属性：

    字体复合属性：

    font: font-style font-weight font-size/line-height font-family;

    • 使用该属性时，必须按上面顺序，不能更改，并且各个属性以空格隔开
    • 不需要设置的属性可以省略，但必须保留font-size和font-family属性，否则font属性则不起作用。
    属性 表示 注意点
    font-size 字号 单位px
    font-family 字体
    font-weight 字体粗细 加粗：700或者bold 不加粗：normal或者400
    font-style 字体样式 倾斜：italic 不倾斜：normal

    文本属性：

    Text属性可定义文本的外观，比如颜色，对齐，装饰，文本缩进，行间距等。

    属性 表示 注意点
    color 通常用十六进制
    text-align 对齐 文字水平对齐方式
    text-indent 缩进 通常段落首行缩进
    text-decoration 文本修饰 下划线：underline 取消：none
    line-height 行高 行与行之间的距离

    css引入方式：

    分为内部样式表，行内样式表，外部样式表。

    外部样式表：

    <link rel= "stylesheet" href="css文件路径">

    复合选择器：

    选择器 作用 特征 使用情况 隔开符号及用法
    后代选择器 用来选择后代元素 可以是子孙后代 较多 符号是空格 .nav a
    子选择器 选择最近一级元素 只选亲儿子 较少 符号是大于 .nav>p
    并集选择器 选择某些相同样式的元素 可以用于集体声明 较多 符号是逗号 .nav,.header
    链接伪类选择器 选择不同状态的连接 跟链接相关 较多 a:hover 鼠标经过
    :focus 选择器 选择获得光标的表单 跟表单相关 较少 input:focus 记住这个写法

    元素显示模式：

    元素模式 元素排列 设置样式 默认宽度 包含
    块级 一行只能放一个 可以设置宽度高度 容器的100% 可以包含任何标签
    行内元素 一行可以放多个 不可以直接设置宽度高度 本身内容的宽度 容纳文本或其他行内元素
    行内块元素 一行可以放多个 可以设置宽度高度 本身内容的宽度

    元素显示模式转换：

    • 转换为快元素：display:block;
    • 转换为行内元素：display:inline;
    • 转换为行内块：display:inline-block;

    CSS背景：

    背景色半透明：

    background: rgba(0, 0, 0, .3);

    属性 作用 值
    background-color
    background-image url()
    background-repeat 是否平铺 repeat/no-repeat/repeat-x/repeat-y
    background-position
    background-attachment 背景附着 scroll(背景滚动)/fixed(背景固定)
    背景简写 颜色 图片地址 平铺 滚动 位置
    半透明

    盒子模型：

    边框(border)

    • border-width 边框粗细

    • border-style 边框样式

      边框样式
      • none 没有边框
      • solid 单实线
      • dashed 虚线
      • dotted 点线

    边框简写

    `border: 1px solid red;

    内边距

    padding: 上 右 下 左 top right bottom left

    外边距：

    margin 简写方式意义跟padding完全一致

    圆角边框：

    border-radius:length:

    简写时分别代表 左上角，右上角，右下角，左下角

    盒子阴影：

    box-shadow: h-shadow v-shadow blur spread color inset;

    文字阴影：

    text-shadow: h-shadow v-shadow blur color;

    浮动

    • float: left 左浮
    • float: right 右浮
    • 浮动的元素会脱离标准流，不再保留原先的位置，会一行内显示并且元素顶部对齐
    • 浮动的元素具有行内块元素的特性
    • 多个盒子浮动 一行内显示并且顶端对齐排列，中间没有缝隙

    清除浮动

    .clearfix:before,.clearfix:after {

    content:"";

    display:table;

    }

    .clearfix:after {

    clear:both;

    }

    .clearfix { *zoom:1; }

    • 父级overflow:hidden;

    定位：

    • 子绝父相
    • 定位叠放次序 z-index 不加单位

    display 属性

    • display: none 隐藏对象
    • display: block 除了转换为块级元素之外，同时还有显示元素的意思
    • display隐藏元素后，不再占有原来的位置

    visibility 可见性

    • visibility :visible 元素可视
    • visibility : hidden 元素隐藏
    • visibility 隐藏元素后，继续占有原来位置

    溢出的文字省略号显示：

    单行文字：

    /*1. 先强制一行内显示文本*/

    white-space: nowrap; （ 默认 normal 自动换行）

    /*2. 超出的部分隐藏*/

    overflow: hidden; /

    *3. 文字用省略号替代超出的部分*/

    text-overflow: ellipsis;

    多行文字：

    overflow: hidden;

    text-overflow: ellipsis;

    /* 弹性伸缩盒子模型显示 */ *

    *display: -webkit-box;*

    */*限制在一个块元素显示的文本的行数 */ *

    *-webkit-line-clamp: 2; *

    */* 设置或检索伸缩盒对象的子元素的排列方式 */

    -webkit-box-orient: vertical;
